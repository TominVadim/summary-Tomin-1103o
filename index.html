Оглавление

Основы HTML

HTML (HyperText Markup Language) — язык для разметки гипертекста, он является набором правил, по которым браузер отличает заголовки от списков, таблицы от картинок и так далее. HTML появился в 1993 году и был призван стандартизировать правила для вывода текста внутри веб-страниц.

Основы CSS

Для визуального оформления WEB-страницы создан язык CSS. CSS переводится как каскадные таблицы стилей (Cascading Style Sheets). Именно этот язык отвечает за то, как наши HTML-элементы будут выведены пользователю в браузере.

Основы JS

JavaScript — один из самых широко используемых языков. Он применяется в разработке игр, создании Web-приложений.

GitHub

GitHub — веб-сервис, который основан на системе Git. Это такая социальная сеть для разработчиков, которая помогает удобно вести коллективную разработку IT-проектов. Здесь можно публиковать и редактировать свой код, комментировать чужие наработки, следить за новостями других пользователей.

Linux

Дистрибутив Linux - это полноценная операционная система, основанная на ядре Linux и включающая приложения, утилиты, графические интерфейсы и другие компоненты, необходимые для работы компьютера или сервера. Однако, Linux-дистрибутивы могут отличаться друг от друга по функциональности, набору программ и внешнему виду.
